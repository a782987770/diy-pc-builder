# 🖥️ 壹米装机大师 — DIY电脑装机配置平台

> 由「**壹米说电脑**」公众号出品的专业电脑DIY装机配置平台  
> 浏览用户分享的装机方案，获取灵感打造你的专属电脑，支持多平台比价和CPS佣金导购

## ✨ 功能特性

### 已完成（MVP v1.0）

| 模块 | 功能 | 状态 |
|------|------|------|
| **装配广场** | 浏览用户分享的装机方案，支持热门/最新/价格/点赞排序 | ✅ |
| **配件库** | 72条配件数据（CPU/主板/内存/硬盘/显卡/电源/机箱/散热器） | ✅ |
| **DIY装配** | 用户自主选配硬件，实时计算总价和跑分 | ✅ |
| **兼容性检查** | 自动检测CPU接口、内存类型、电源功率、显卡长度、散热器高度 | ✅ |
| **CPS佣金链接** | 每个配件关联京东/淘宝/拼多多推广链接，显示佣金比例 | ✅ |
| **方案详情页** | 完整的配件清单表格 + 购买跳转 + 点赞功能 | ✅ |
| **响应式设计** | 支持PC端和移动端访问 | ✅ |

## 🚀 快速开始

### 环境要求

- Node.js >= 18.0
- npm >= 9.0

### 安装运行

```bash
# 1. 进入项目目录
cd diy-pc-builder

# 2. 安装依赖（已完成可跳过）
npm install

# 3. 初始化数据（合并分散的JSON为统一parts.json）
npx tsx lib/init-data.ts

# 4. 启动开发服务器
npm run dev

# 5. 浏览器访问
open http://localhost:3000
```

## 💰 CPS佣金模式说明

### 当前配置的佣金比例示例

| 平台 | 通用比例 | 电脑数码类 | 高佣活动期 |
|------|----------|------------|-----------|
| 京东联盟 | 3%–5% | 3%–25% | 上浮15% |
| 淘宝联盟 | 2%–4% | 1%–20% | 大促更高 |
| 拼多多 | 4%–6% | 可达30%+ | – |

### 如何替换为真实CPS链接

1. 注册 [京东联盟](https://union.jd.com/) / [淘宝联盟](https://www.alimama.com/)
2. 获取推广位PID和推广链接
3. 在 `data/parts-*.json` 中搜索 `cpsLinks` 字段
4. 将示例URL替换为你的真实推广链接：
```json
{
  "platform": "jd",
  "url": "https://item.jd.com/真实商品ID.html?utm_source=你的渠道标识",
  "commissionRate": 0.03,
  "priceAtPlatform": 实际价格
}
```

## 📊 兼容性检查规则

系统自动检测以下兼容性问题：

| 检测项 | 严重级别 | 说明 |
|--------|----------|------|
| CPU ↔ 主板接口 | ❌ 错误 | AM5/LGA1700/LGA1851等必须匹配 |
| 内存 ↔ 主板类型 | ❌ 错误 | DDR4/DDR5必须匹配主板支持 |
| 电源功率 | ⚠️ 警告 | 预估功耗超过额定85%时提醒 |
| 显卡长度 ↔ 机箱 | ❌ 错误 | 显卡物理尺寸不能超过机箱限长 |
| 散热器高度 ↔ 机箱 | ❌ 错误 | 散热器高度不能超过机箱限高 |
